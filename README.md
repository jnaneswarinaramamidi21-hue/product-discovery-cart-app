# Product Discovery & Cart Web App

A simple frontend web app to browse products, search, filter, and manage a shopping cart.  
Built with **HTML, CSS, and JavaScript**, using **LocalStorage** to save cart items.

## Features
- Product listing with search and filter
- Sort by price
- Add and remove products in cart
- Persistent cart using LocalStorage

## How to Run
1. Open `index.html` in a browser
2. Add products to cart and view them in `cart.html`
